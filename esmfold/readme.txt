Para correr los experimentos con esm-fold:

1. Ejecutar el script de bash setup_env.sh, que crea un entorno virtual de python, e instala las librerías requeridas.

2. Ejecutar el script de bash run_all_esmfold.sh, que ejecuta de manera secuencial los experimentos para cada .fasta que se encuentra en las carpetas Novel y Similar. Esto lo hace a través de llamadas al script de python run_esmfold.py
